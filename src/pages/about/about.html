Acerca del proyecto

Una web para explorar y reproducir canales IPTV

Este sitio fue creado para consultar canales IPTV de forma simple, visual y ordenada. Permite descubrir canales por categoria, cargar contenido destacado como anime y abrir cada transmision en un reproductor integrado dentro de la misma experiencia.

live_tv

Sobre la web

Objetivo y funcionamiento

La idea de esta aplicacion es ofrecer una interfaz mas amigable para navegar listas IPTV publicas, mostrar la informacion esencial de cada canal y facilitar la reproduccion sin tener que revisar manualmente archivos M3U.

link

Fuente de canales

Origen de la lista IPTV

Los canales mostrados en esta web se obtienen desde la playlist publica del proyecto IPTV-org, utilizada como fuente de datos para cargar y clasificar el catalogo.

Tecnologias usadas

Base tecnica del proyecto

  • check_circle Angular 21 para la estructura de la aplicacion y el enrutamiento.
  • check_circle TypeScript y RxJS para la logica, consumo y transformacion de datos.
  • check_circle Angular Material y Tailwind CSS para construir la interfaz visual.
  • check_circle hls.js para mejorar la reproduccion de transmisiones compatibles.

Sobre el desarrollador

Proyecto desarrollado como practica de frontend moderna

Esta web fue desarrollada por Ricardo Aguilar, como una forma de practicar integracion de APIs, procesamiento de listas M3U, organizacion de componentes y construccion de interfaces modernas con Angular. Tambien funciona como proyecto de portafolio para demostrar manejo de arquitectura frontend, consumo de datos remotos y experiencia de usuario.
